#  Retro Arcade Hub

Hey! Welcome to the Retro Arcade Hub. This is a complete, browser-based arcade machine 

I built this project HTML, CSS, and  JavaScript. The website have high scores, achievements, and profiles.

##  What is this?

In the  dashboard where you can play 5 classic games, customize your profile, unlock achievements, view stats, and customize the theme 

The games:

*   **Snake**
*   **Pong** 
*   **Tic Tac Toe**
*   **Memory Match**
*   **Rock Paper Scissors**



###  Sound effects
I wrote an 8-bit sound generator code in `js/app.js` using the browser's native web Audio API. 


### Database
Since there's no backend database, all the details, custom themes, default settings, game high scores, and achievement  are saved in your browser's local storage. Your progress is saved automatically and stay even if you close the tab or refresh the page.

###  Themes
You can change the  arcade skin under settings. I designed 5 themes using CSS variable mapping:
1.  **Space Cadet**
2.  **whiteboard**
3.  **Retro Neon**
4.  **Cyberpunk**
5.  **Matrix Terminal**


Have fun!
